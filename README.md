# Nuredin Kassaw - Portfolio

A modern, responsive personal portfolio built with **React**, **Vite**, and **Tailwind CSS**.

## 🚀 Features

- Responsive design for mobile, tablet, and desktop
- Light/dark theme toggle
- Sections for About and Projects
- Featured project cards with technology tags and external GitHub links

## 🛠️ Tech Stack

- [React 18](https://react.dev/)
- [Vite 5](https://vitejs.dev/)
- [Tailwind CSS 3](https://tailwindcss.com/)
- ESLint for code linting

## 🏁 Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
```

### Run in development

```bash
npm run dev
```

Then open the local URL shown in your terminal (default: `http://localhost:5173`).

### Build for production

```bash
npm run build
```

### Lint the code

```bash
npm run lint
```
